## üöÄ Installation & Setup

### 1. Clone the Repository
```bash
git clone [https://github.com/Yogaprtamaa/time-tracker-widget.git](https://github.com/Yogaprtamaa/time-tracker-widget.git)
cd time-tracker-widget

2. Install Dependencies
npm install

3. Configure API Keys (Important) üîí
This project uses a secrets.js file to store Supabase credentials securely. This file is ignored by Git. You must create it manually.

Create a file named secrets.js in the root folder (next to index.html).

Paste the following code inside it:

JavaScript

// secrets.js
const SUPABASE_CONFIG = {
    URL: 'YOUR_SUPABASE_URL_HERE',
    KEY: 'YOUR_SUPABASE_ANON_KEY_HERE'
};
(Replace YOUR_SUPABASE_URL_HERE and KEY with your actual Supabase credentials).

4. Database Setup (Supabase)
To make the logging work, go to your Supabase SQL Editor and run this query to create the table:

SQL

create table logs (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  activity text,
  duration text
);
5. Run the App
Bash

npm start
